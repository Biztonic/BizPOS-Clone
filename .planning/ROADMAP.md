# BizPOS Roadmap

This roadmap tracks the evolution of the Biztonic POS system.

## Completed Milestones

### [x] Phase 1: Core POS Engine
- Implementation of basic order management and inventory logic.
- SQLite-based local storage setup.
- Print service foundation.

### [x] Phase 2: Multi-platform & UI
- Expansion to Android, iOS, and Windows.
- Implementation of the generic theme and design system.
- Customer management and loyalty foundation.

### [x] Phase 3: Cloud Sync & Event Sourcing
- Integration of Firebase Firestore as the authoritative backend.
- Implementation of the Ledger-based sync strategy.
- Multi-store isolation and role-based access control.
